
## Installation

### Ubuntu 

```bash
$> sudo apt install libpcl-dev
$> cd ~
$> git clone https://github.com/omar-bakr/Road-segmentation.git
$> cd Road-segmentation
$> mkdir build && cd build
$> cmake ..
$> make
$> ./environment
```
